# 僵尸毁灭工程直播描述优化

## 受众画像分析

### 核心受众特征

| 维度 | 描述 |
|------|------|
| **年龄层** | 25-40岁为主力 |
| **性别** | 男性为主(约80-85%) |
| **游戏偏好** | 生存沙盒、模拟经营、硬核游戏 |
| **付费意愿** | 高 - 愿意购买mod、服务器、游戏 |
| **观看时段** | 晚间(20:00-24:00)、周末 |
| **内容偏好** | 生存技巧、联机互动、灾难应对、mod演示 |

### 受众细分

1. **硬核生存玩家** - 追求高难度、极限生存
2. **建造规划型** - 喜欢基地设计、长期规划
3. **联机社交型** - 看重多人互动、剧情
4. **mod探索者** - 追求新内容、汉化
5. **云玩家/新手** - 学习生存攻略
